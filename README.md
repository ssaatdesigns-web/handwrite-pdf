# Text PDF → Handwritten PDF (Vercel-ready)

## What it does
- Upload a **text-based PDF**
- Choose:
  - Paper: blank / single ruled / multi ruled
  - Border on/off + thickness + margin
  - Pen color: red/blue/black/custom hex
  - Handwriting style (font)
  - Font size
  - Output filename
- Download a new **handwritten-style PDF**

## Setup (local)
```bash
npm install
npm run dev
